A static launcher page for common AI chat services.

## What lives here

- `index.html` builds the grid.
- `icons/` holds the tile assets.

## Community

- Use [Discussions](https://github.com/yookibooki/chatbots/discussions) for ideas and broader feedback.
- Use Issues for broken links, missing icons, or layout bugs.
- Keep PRs small and asset-only when possible.

## Local preview

Serve the folder however you like, for example:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
