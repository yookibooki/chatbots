# Contributing

Thanks for helping improve Chatbots.

## Before you open a PR

- Check [Discussions](https://github.com/yookibooki/chatbots/discussions) and Issues first.
- For UI or layout changes, include before/after screenshots or a short screen recording.
- Keep `index.html` readable and avoid unnecessary complexity.
- New tiles should have a working link and a matching icon in `icons/`.

## Suggested workflow

1. Open a Discussion for a new idea.
2. File an Issue for a bug or broken link.
3. Submit a PR with the smallest change that solves it.

## Review checklist

- Links work
- Icons load
- Layout still fits on common screen sizes
- No unrelated changes
