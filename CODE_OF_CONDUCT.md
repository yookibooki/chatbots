# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful
- Be constructive
- Assume good intent
- Keep feedback focused on the work

## Unacceptable behavior

- Harassment
- Hate speech
- Personal attacks
- Doxxing or threats

## Enforcement

If you need moderation help, open a Discussion and ask for maintainer review.
